Función Haversine

calcular_distancias <- function(df_input, lat_ref = 4.7110, lon_ref = -74.0721) {
  R <- 6371.0
  lat1 <- lat_ref * pi / 180
  lon1 <- lon_ref * pi / 180
  lat2 <- df_input[["Latitud"]] * pi / 180
  lon2 <- df_input[["Longitud"]] * pi / 180
  dlat <- lat2 - lat1
  dlon <- lon2 - lon1
  a <- sin(dlat / 2)^2 + cos(lat1) * cos(lat2) * sin(dlon / 2)^2
  c_dist <- 2 * atan2(sqrt(a), sqrt(1 - a))
  df_input[["Distancia_Bogota"]] <- round(R * c_dist, 2)
  return(df_input)
}

df_dist <- calcular_distancias(df)
print(df_dist[, c("Ciudad", "Distancia_Bogota")])

1       Bogotá             0.00
2     Medellín           238.67
3         Cali           306.67

Ejercicio 2: Limpieza y agregación tabular

Paso 1: DataFrame con nulos

df_est <- data.frame(
  Estacion      = c("Páramo", "Valle", "Costa", "Selva", "Sabana"),
  Elevacion     = c(3200, 1500, 15, 200, 2600),
  Precipitacion = c(850.5, 1200.0, NA, 3000.5, NA)
)
print(df_est)

Paso 2: Imputación

library(dplyr)
media_p <- mean(df_est[["Precipitacion"]], na.rm = TRUE)
cat(sprintf("Media válida: %.2f mm\n", media_p))
Media válida: 1683.67 mm
df_est <- df_est %>%
  mutate(Precipitacion = ifelse(is.na(Precipitacion), media_p, Precipitacion))
print(df_est)

Comentarios

En R se usa na.rm = TRUE en mean() porque sin este argumento retorna NA si hay nulos. Se eligió dplyr filter por legibilidad con pipes. Las funciones trigonométricas en R son vectorizadas nativamente. La imputación con media es la técnica más simple; en SIG real se usaría interpolación espacial.
